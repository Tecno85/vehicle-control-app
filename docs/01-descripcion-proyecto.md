# Descripción del proyecto

## Nombre del proyecto

VehicleControlApp

## Nombre visible de la aplicación

Control Vehicular

## Objetivo

Desarrollar una aplicación móvil Android usando Kotlin que permita llevar el control de vehículos particulares y taxis, registrando gastos, novedades, fechas importantes y recordatorios.

## Descripción

Control Vehicular será una aplicación móvil diseñada para administrar inicialmente cuatro vehículos:

- Dos taxis.
- Dos carros particulares.

La aplicación permitirá registrar gastos, novedades y fechas importantes asociadas a cada vehículo, como el vencimiento del SOAT, la revisión tecnicomecánica y los impuestos.

En el caso de los taxis, la aplicación también permitirá manejar un ingreso fijo diario y ajustar ese ingreso cuando exista una novedad que afecte la operación, como una falla mecánica, día de descanso o ausencia del conductor.

## Alcance inicial

La primera versión de la aplicación permitirá:

- Registrar vehículos.
- Diferenciar entre taxis y carros particulares.
- Configurar ingreso fijo diario para taxis.
- Registrar gastos por vehículo.
- Registrar novedades por vehículo.
- Registrar vencimiento del SOAT.
- Registrar vencimiento de la revisión tecnicomecánica.
- Registrar fechas relacionadas con impuestos.
- Mostrar recordatorios internos dentro de la aplicación.
- Calcular balance diario para los taxis.

## Fuera del alcance inicial

En la primera versión no se incluirá:

- Inicio de sesión.
- Sincronización en la nube.
- Firebase.
- Exportación a PDF o Excel.
- Notificaciones avanzadas del sistema Android.
- Administración de usuarios.
